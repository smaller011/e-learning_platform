Selectors

Selectors are used to “pick” elements you want to style.

Lesson notes

Selectors choose which elements to style, from simple tags to advanced combinations for precise control. You can target tags, classes, IDs, or even combine rules for more control.

  • Element, class (.class), ID (#id)
  • Descendant/child, attribute selectors
  • Specificity & cascade basics
