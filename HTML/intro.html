Introduction To HTML

This lesson introduces you to the basics of HTML.

Lesson notes

HTML (HyperText Markup Language) is the standard language for creating web pages. It uses elements like <h1>, <p>, and <a> to structure content.

  • HTML stands for HyperText Markup Language
  • It is used to create the structure of the webpage
  • Works with CSS and JavaScript
