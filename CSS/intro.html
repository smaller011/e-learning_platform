Introduction To CSS

CSS (Cascading Style Sheets) is used to style and design web pages by controlling colors, layouts, and fonts.

Lesson notes

CSS adds design to your web pages. It controls layout, colors, and fonts, turning plain HTML into attractive and user-friendly websites.

  • Ways to add CSS: inline, internal, external
  • Basic syntax: selector { property: value; }
  • Works well with HTML
