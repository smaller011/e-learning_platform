HTML Elements

This lesson explains the basic building blocks of HTML and how elements are used to structure a webpage

Lesson notes

HTML elements are the building blocks of a webpage. They usually come in pairs: an opening tag and a closing tag.

  • Elements are defined by tags, e.g. <h1>Hello</h1>
  • Some elements are empty (self-closing), e.g. <br> <img>
  • Elements can be nested inside one another
