Flexbox

Flexbox is a layout system that arranges items in rows or columns. It makes aligning, centering, and spacing elements easier.

Lesson notes

Lay out items in rows or columns and align them easily, perfect for navs, cards, and toolbars. Flexbox is a layout model that makes it easier to arrange items in rows or columns. It allows you to align, space, and distribute elements evenly without using float or complex positioning.

  • justify-content, align-items, gap
  • flex-wrap, order, flex shorthand
