Variables and Data Types

Variables store information, and data types (like numbers, strings, and booleans) define the kind of data you can use.

Lesson notes

Variables are used to store data, and JavaScript supports different data types.

  • Variables: var, let, and const.
  • Data Types:
    • Numbers (10, 3.14)
    • Strings ("Hello")
    • Booleans (true, false)
